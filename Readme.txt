Read this !!

1.First create a database named timecapsule in phpmyadmin console if you use MySql from xampp server

2.Import the timecapsule.sql file provided in this folder using the import tab of your database in phpmyadmin.

3.Load project using Eclipse or required IDE.

4.Now run the project as Run on server(Server should be added to Eclipse).

NOTE: Start MySql database before launching the application.
